# Restaurant Billing System
This is a Python-based Restaurant Billing System developed using Tkinter for the graphical user interface. The application allows users to select food items, enter customer details, generate bills, and save the billing information automatically.

## Features
- GUI based billing system using Tkinter
- Customer name and mobile number input
- Menu with selectable food items
- Quantity selection for each item
- Automatic bill generation
- Bill saved as a text file
- Search previously generated bills

## Technologies Used
- Python
- Tkinter
- File Handling
- OS Module

## How to Run the Project
1. Make sure Python is installed on your system.
2. Run the program using:

   python restaurant_billing_system.py

3. The billing system window will open.
## Project Functionality
1. Enter customer name and mobile number
2. Select food items and quantities
3. Click Generate Bill
4. Bill will be displayed and saved in the bills folder
